#C语言游戏（烂尾）
学了一学期C语言，想着用C语言+EasyX开发一个游戏，由于个人水平有限，程序在交替加载动画的时候会一直闪烁，尝试了好几天没有解决，决定放弃这个项目，转学unity。

目前已经实现了角色的移动，背景的移动，随机刷怪，攻击，击倒敌人获得金币等功能
